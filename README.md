# CSM-Modsets-SimWaveform

基于 [CSM（可通信状态机）](https://nevstop-lab.github.io/CSM-Wiki/) 框架的模拟波形生成模块，用于在无实际采集硬件时提供测试波形数据。

This repository contains a CSM-based module for generating simulated waveform data for development and testing purposes.

---

## 模块列表

| 模块 | 所属模块组 | 功能说明 |
| --- | --- | --- |
| [WaveformGen](./WaveformGen.md) | CSM-SimWaveform.lvlib | 模拟波形数据生成，支持配置波形类型、幅值、频率等参数 |
